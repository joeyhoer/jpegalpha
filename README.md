# jpegalpha

`jpegalpha` converts a PNG to an SVG with embedded JPGs (the image and an alpha mask). This reduces filesize significantly.

## Usage

~~~
Usage:     jpegalpha.sh [OPTIONS]
Version:   1.0.0

Converts a PNG to an SVG with embedded JPGs (the image and an alpha mask).

OPTIONS:
  -q  Image quality [default: 80]
  -m  Mask quality [default is equal to image quality]

~~~
